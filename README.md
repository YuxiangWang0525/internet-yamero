# Internet YAMERO
> 在简中互联网上保护自己思想不被某些不明生物糟蹋的屏蔽词表
## 为什么有这个项目
由于网民入网门槛过低,目前简体中文互联网上会出现一堆不知道那里没尽职看守导致发出了一堆一言难尽的评论、作品  
好听一点:抽象, 难听一点:无可救药  
众所周知,在一个素质参差不齐的互联网上你永远没办法看到一片祥和(曾经的B站有,结果因为某些原因全搅和了,呵呵)  
那既然没有能替代他们的平台,也只能我们自己出手手动规避,以退为进了。
**词库比较主观  这个项目仅仅是屏蔽部分涉政内容。我不会要求任何人强行站队,但为了我自己的未来和安全以及这个项目在中国内地的正常使用,我不会加一些激进词汇。**
## 这个项目该怎么使用
很遗憾的说,我们配套的工具**ClearFlow**还处于Dev阶段。很多网站无法即使适配
如果你要用做屏蔽词列表,请注意屏蔽带#的注释
